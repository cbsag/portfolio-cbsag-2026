## Overview
Build a biomedical QA system where each generated claim remains traceable to retrieved evidence.

## System design
- Retrieval stack: BM25/TF-IDF and MedCPT dense retrieval
- Generator stack: Qwen-based models served via vLLM
- Multi-stage context building with MMR-style selection and reranking

## What was built
- End-to-end pipeline for BioGen Task-B submissions
- Citation-linked answer generation with PubMed evidence traces
- Interactive results dashboard for run inspection and comparison

## Results
- Reproducible run artifacts and diagnostics for retrieval/answer quality
- Better transparency for evidence verification during evaluation
